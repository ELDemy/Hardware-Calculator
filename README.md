# Calculator
## A proteus simulation for a calculator using logic circuits.

|  Operation  |       Description       |
|  ---------  |       -----------       |
|      +      |  Output <- N1 + N2      |
|      -      |  Output <- N1 - N2      |
|      *      |  Output <- N1 * N2      |
|      /      |  Output <- N1 / N2      |
|      M+     |  M <- M + N1            |
|             |  Output <- M            |
|      M-     |  M <- M - N1            |
|             |  Output <- M            |
|      MC     |  M <- 0                 |
|             |  Output <- M            |
